# PTCG Deck Agent 0.4.1 Release Notes

Release date: 2026-06-13

## Main Change

0.4.1 is the current client release. It keeps the Simplified Chinese PTCG environment content bundled in-game and updates the runtime/export version metadata for all supported platforms.

## Included Content

- Added 58 new cards for the Simplified Chinese 17.0 environment.
- Added 12 bundled player decks modeled after the international environment.
- Added 7 bundled AI decks for direct practice and AI opponent selection.
- Manual update checks bypass HTTP/CDN caches so mobile clients do not keep reading stale update manifests.

## Compatibility Notes

- App runtime version: `0.4.1`
- Build number: `41`
- Windows export version: `0.4.1.0`
- macOS short version: `0.4.1`
- Android version name/code: `0.4.1` / `41`

## Scope

This release is content-environment focused. The release goal is to let players enter the 17.0 card pool and deck environment directly after installation, with the new bundled cards, player decks, and AI decks available in-game.
